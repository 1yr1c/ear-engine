ML Modules
Module 1 — OLS Regression
→ Pass-through rates in EaR formula
Replaces static Fabra & Reguant (2014) sector rates with rates estimated from EU ETS price history vs actual gross margins. Higher R² = more empirically grounded EPS impact.
Module 2 — ARIMA Forecasting
→ Emissions intensity per holding
Projects each holding's Scope 1+2 emissions intensity forward 3 years using ARIMA(1,1,0) rather than using static 2023 figures. Captures decarbonisation trajectories already underway.
Module 3 — Gaussian HMM
→ Default scenario selection
Detects current EUA carbon market regime (low volatility / shock / spike) from monthly log returns. Maps detected regime to the most appropriate NGFS scenario on load.
Module 4 — scipy SLSQP
→ Portfolio optimiser weights
Mean-variance optimisation minimising λ·EaR + (1-λ)·w'Σw subject to turnover constraint. Uses single-factor covariance matrix (beta + EaR-scaled idiosyncratic risk) to prevent concentration.
